# codealpha_tasks--A-Mini-Project-Report-on-Computer-Aided-Drug-Design-CADD-
The Standard Drug Discovery Pipeline
1. Target Identification & Validation → 2. Hit Discovery → 3. Hit-to-Lead Optimization → 4.
Preclinical Testing (ADMET) → 5. Clinical Trials (Phases I-III) → 6. Regulatory Approval.
Bioinformatics intervenes heavily in phases 1 through 4, drastically reducing the time and
compounds needed for physical testing.
